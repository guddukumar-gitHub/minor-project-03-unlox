# Weather App & Movie Search App

A responsive web application consisting of two interactive projects: a **Weather App** and a **Movie Search App**, built using **HTML5, CSS3, and JavaScript (ES6+)**. The applications integrate public REST APIs to fetch and display real-time weather and movie information while demonstrating asynchronous programming, dynamic DOM manipulation, responsive UI design, and robust error handling.

## Features

### 🌤 Weather App
- Search weather by city name
- Display current temperature and weather conditions
- Show humidity and wind speed
- Weather icons based on current conditions
- Error handling for invalid city names
- Responsive user interface

### 🎬 Movie Search App
- Search movies by title
- Display movie posters and details
- Show IMDb rating, release date, runtime, genre, and plot
- Fallback image for missing or broken posters
- Error handling for invalid movie searches
- Responsive card-based layout

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- Async/Await
- ES Modules
- OpenWeather API
- OMDb API
